Hexo FexO主题中加入网页音乐播放

2019.04.26 Linkenwild hexo美化  热度 ℃

hexo添加网页音乐播放

download Aplayer 源码

访问地址

使用git clone下载到本地，

git clone https://github.com/MoePlayer/APlayer

解压后将dist文件复制到themes\fexo\source下

新建music.js
