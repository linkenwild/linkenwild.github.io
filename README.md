这里是我个人的小天地，主要用于每天学习知识的总结，以及研究生生涯的点滴记录。

鱼的记忆只有七秒，人类的记忆有多少时间呢？这里是个未知数，把博客作为知识输入和消化输出的集散地，也能为将来的自己的提升加分项。

学生生涯很短暂，也把这里作为学校生涯和社会生涯的衔接过渡的节点。每一步的跳跃都来自于前一步铿锵有力、掷地有声的决策和部署。
