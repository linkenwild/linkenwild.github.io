# 局域网与交换机
___
## 以实验室为例
____
实验室进来的是一根网线，网线连接路由器，从路由器出来一根网线接到交换机端口上，实验室所有的台式主机各使用一根网线连接到交换机端口上。路由器具有开启wifi功能。

交换机ip为192.168.3.1，实验室所有电脑都会分配一个临时ip为192.168.3.xx（x是2-255）。手机连接路由器上的wifi热点，也会随机的分配192.168.3.xx（x是2-255）
